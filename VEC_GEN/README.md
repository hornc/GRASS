# vgasm

Experimental attempt to parse and compile Vector General display instructions
to raw bytes for Vector General DMA.

### purpose
To get a feel for the layer beneath GRASS, and the raw binary memory format GRASS likely used for defining PATH objects.

### sources
* Sample Program: [Vector General Graphics Display System Reference Manual Jan72, 4-1](https://archive.org/details/bitsavers_vectorGenephicsDisplaySystemReferenceManualJan72_6676862/page/n93)
* [3.18 Control Display Instructions 3-14](https://archive.org/details/bitsavers_vectorGenephicsDisplaySystemReferenceManualJan72_6676862/page/n71)
